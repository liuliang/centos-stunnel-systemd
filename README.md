centos-stunnel-systemd
===============================

CentOS 7 systemd (systemctl) file to use with stunnel

Save file to:

    /etc/systemd/system/stunnel.service

Use commands like:

    sudo systemctl start stunnel.service
    sudo systemctl enable stunnel.service
